## Структура сайта — что работает (консенсус)

```
1. HERO        — заголовок + CTA + 3 счётчика (лет / клиентов / % одобрений)
2. УСЛУГИ      — визы / туры / ВНЖ / доп. услуги (цены "от...")
3. КАК РАБОТАЕМ — 4-5 шагов (заявка → консультация → документы → виза)
4. ПРЕИМУЩЕСТВА — 4-6 карточек (опыт, гарантия, местный центр)
5. СТРАНЫ      — сетка популярных направлений
6. ОТЗЫВЫ      — 6+ отзывов с именем/фото + рейтинг Яндекс/Google
7. FAQ         — 7-10 вопросов
8. КОНТАКТЫ    — адрес Петрозаводск + форма + карта
```

## УТП — варианты для "Гражданина мира"

**Лучшие варианты hero-заголовка:**
- "Визы, туры и ВНЖ — всё в одном месте в Петрозаводске"
- "Виза в Европу за 3 дня — без беготни и очередей"
- "Получи визу без отказа — или вернём деньги"
- "От визы до тура — помогаем на каждом шагу"

**Триггеры доверия:**
- Счётчики: X лет / X клиентов / X% одобрений
- Отзывы с фото и именем
- Фото офиса и команды (живые люди!)
- Рейтинг на Яндекс.Картах и Google
- Гарантия возврата денег при отказе
- Раздел "В СМИ" (если есть публикации)

## Работа с ценами

**Рекомендация:** показывать цену "от X ₽" по каждому направлению.
- Полные цены скрыты — люди уходят к конкурентам
- Цены "от..." привлекают + оставляют пространство для консультации
- Отдельная страница/секция с таблицей цен — хорошо для SEO

## SEO-стратегия (без блога почти никто не делает — точка роста!)

**Ключевые запросы для Петрозаводска:**
- "виза в Финляндию Петрозаводск" (топ!)
- "шенген Петрозаводск"
- "визовый центр Петрозаводск"
- "ВНЖ Финляндия помощь"
- "виза в [страну] Петрозаводск"

**Страница на каждую страну** = основа SEO для ниши.
Пример: `/viza-finlyandiya`, `/viza-italiya`, `/vnzh-ispaniya`

## Фишки для конкурентного преимущества

1. **Открытые цены** — большинство скрывает, мы показываем → доверие
2. **Близость к Финляндии** — сделать финскую визу флагманской услугой
3. **Живые люди на сайте** — фото команды, видео-отзывы (никто не делает!)
4. **Курьерская доставка** — получить документы дома/в офисе (только visa-prosto.com в Казани)
5. **Telegram/соцсети** — почти никто не ведёт активно → огромная ниша
6. **Блог** — статьи "как получить финскую визу", "ВНЖ в 2026" дают SEO-трафик
