## Инструменты продвижения конкурентов (что проверить)

- [ ] Яндекс.Карты — есть у некоторых

- [ ] Telegram — нужно проверить вручную

## Где "Гражданин мира" может выиграть
1. **Visavia рейтинг 3/5** — их клиенты недовольны → забираем
2. **У большинства нет нормального сайта** → наш сайт станет выгодным отличием
3. **Мало кто ведёт соцсети** → Telegram-канал с контентом даст преимущество
4. **ВНЖ + туры** — редко кто предлагает комплекс → УТП "визы · туры · ВНЖ" рабочее
